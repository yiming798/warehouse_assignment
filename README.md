# warehouse_assignment

A blank repository ready for importing existing Java code.

## Getting Started

Import your existing Java source files into this repository and push them here.

## Project Structure

Place your Java source files under a suitable directory structure, for example:

```
src/
  main/
    java/
      com/example/
        *.java
  test/
    java/
      com/example/
        *Test.java
```
